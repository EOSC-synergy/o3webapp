Constructor

# <SearchBar inputArray foundIndicesCallback shouldReturnValues />

A searchbar component that is used for searching a string in a data array that either contains strings or objects. If the array contains objects the values of these objects are searched.

PropTypes:
Name Type Required Description Default
inputArray array Yes an array of either only strings or only objects. The input typed in the searchbar is used as a search string. If an array of objects is passed the values of each object are searched.
foundIndicesCallback func Yes this function is called with an array containing the indices that matched the search string. The function is called after the search is performed.
shouldReturnValues bool No specifies whether the searchbar should pass indices or values to props.foundIndicesCallback
Parameters:
Name Type Description
props Object
