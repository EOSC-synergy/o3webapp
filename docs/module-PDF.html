# static downloadGraphAsPDF(plotId, fileName, modelGroups, currentData)

Downloads the PDF which contains the Graph in SVG format and contains the List of models.

Parameters:
Name Type Description
plotId

the plot id of the graph (tco3_zm, tco3_return etc.)
fileName

the File name of the PDF
modelGroups

the Model Groups which contains the names of the models
currentData

the current Model data contains the properties of the models (color, line style, etc.)

# inner getAdjustedSVG(svgElement)

This Method adjusts the svg element in order to scale it right in the pdf file. the viewBox parameter of the svg element will be set to the width and height values of the svg element. visit the following website for more details about the viewBox parameter: https://developer.mozilla.org/en-US/docs/Web/SVG/Attribute/viewBox
