Module

API

This module is responsible for the communication with the API and it handles the fetching of the relevant data. The fetched data is then applied to the corresponding element in the redux store. It sets up and implements all relevant reducers for the interchange with the API and manages the request state of the webapp.

# static getPlotData(plotId, latMin, latMax, months, modelList, startYear, endYear, refModel, refYear)

Performs a request to /plots/plotId to fetch the plot data from the API formatted with the given parameters.

Parameters:
Name Type Description
plotId string

a string describing the plot - has to be the official plot name (e.g. tco3_zm)
latMin int

specifies the minimum latitude
latMax int

specifies specifying the maximum latitude
months Array:.<int:>

represents the selected months
modelList Array:.<String:>

lists the desired models
startYear int

from which point the data should start
endYear int

until which point the data is required
refModel string

the reference model to "normalize the data"
refYear int

the reference year to "normalize the data"
